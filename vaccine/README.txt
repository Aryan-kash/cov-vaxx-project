The tech stack for this project is flask and sqlalchemy
XAMPP is used to host the database server
Install python and XAMPP before running
also install the following with the given commands:
install flask
install mysqlclient
install flask sql alchemy
install flask login

admin credentials are given in config.json 
import sql file in the xamp phpmyadmin

The full explanation on how to run the project given in the README2.txt inside the project
